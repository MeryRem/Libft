# Libft

Librería en C que contendrá una serie de funciones de propósito general.
Programar en C puede ser tedioso cuando no se tiene acceso a las funciones estándar más utilizadas. Este proyecto tiene como objetivo ayudar a comprender cómo actuán estas funciones, 
implementarlas y aprender a utilizarlas de forma eficaz.
Desarrollarás una librería propia que te será muy útil en los próximos proyectos de C.
