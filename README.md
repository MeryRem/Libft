# Libft

Librería en C que contendrá una serie de funciones de propósito general que tus programas podrán utilizar.
Programar en C puede ser tedioso cuando no se tiene acceso a las funciones estándar más utilizadas. Este proyecto tiene como objetivo ayudarte a comprender cómo actuán estas funciones, 
implementarlas por tu cuenta y aprender a utilizarlas de forma eficaz.
Desarrollarás una librería propia que te será muy útil en los próximos proyectos de C.
